## 💡 避雷与建议
1.  **时间规划：** 最佳备考期 **2-3个月**，集中精力突击比拉长战线更有效。
2.  **关于报班：** 网课资料丰富，自制力强者 **无需报班**。
3.  **心态：** 拒绝“5天速成/小学生词汇逆袭”等歪门邪道，脚踏实地。
